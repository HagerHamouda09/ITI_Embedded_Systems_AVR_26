# C Programming Foundations
> **Preparation Module for AVR Embedded Training**

---

## Sessions Checklist

*   **Session 1:** C Basics (Variables, Data Types & Operators)
*   **Session 2:** Conditional Statements (`if`, `switch-case`)
*   **Session 3:** Loops (`for`, `while`, `do-while`, `break`, `continue`)
*   **Session 4:** Functions (Prototypes & Architecture)
*   **Session 5:** Pointers & Direct Memory Access 
*   **Session 6:** Data Modifiers & Storage Classes (`static`, `volatile`)
*   **Session 7:** User-Defined Data Types (`struct`, `union`, `enum`, Bit-fields)
*   **Session 8:** C Build Process & Preprocessor Directives (`#define`, Conditional compilation)
*   **Session 9:** RAM Layout & Dynamic Memory Allocation (`malloc`, `free`, Linked Lists)
